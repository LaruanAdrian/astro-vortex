# Astro Vortex

**Astro Vortex** is a 2D space shooter where you battle alien forces inside a collapsing space-time vortex.

## 🎮 Controls

- Arrow ↑ ↓ — Move your ship
- Spacebar — (Planned) Fire weapon

## 🚧 In Development

- Enemy AI
- Boss levels
- Power-ups
- Sound & visuals
